PROJECT:
    Een website om aan te geven hoe je je dag hebt besteed, en hoe goed je je daarbij voelde. uiteindelijk krijg je een score.
    dit is op de schaal van een dag, maar kan ook op de schaal van een week zijn. (als het mogelijk is zelfs een maand, maar dat is nogal veel).



    | WAT PAS IK TOE |: 
        - PHP
        - HTML
        - SQL
        - Javascript
        - CSS of Bootstrap


    | ONDERDELEN VAN WEBSITE |:
        - Een Voorpagina
        - Een Account registratie / inlogpagina
        - Een index pagina (details onder tussenkop 'details')
        - Een dagmeter pagina (om te kijken hoe je dag is gegaan aan de hand van wat je invult)
        - Een terugblik deel (om te kijken hoe je week is gegaan)


    | MOGELIJKE ONDERDELEN VAN WEBSITE |:
        - Een Instellingen pagina
        - Een Uitbreiding van de terugblik pagina (voor een terugblik van een maand)




    | DETAILS |:
        voorpagina:
            een pagina om een nieuwe bezoeker te ontvangen en uitleggen hoe de website werkt en het doel ervan.
        
        account registratie / inlogpagina:
            spreekt voor zich

        index:
            een overzicht, deze pagina bevat het terugblik deel. dit is de hoofdpagina. 
            
            er is een snel toegankelijke knop om naar de dagmeter pagina te gaan.

            nav balk bovenaan met account knoppen, koppelingen naar andere pagina's en dergelijke.

        dagmeter pagina:
            een pagina waar je invult hoe je dag ging, wanneer je deze data invult en op de klaar knop klikt, geeft de pagina een leuk overzicht van het volgende:
                - de tijd verdeling van je dag
                - wat je vandaag hebt bereikt
                - hoe je voelt over de dag
                - een score van vandaag
